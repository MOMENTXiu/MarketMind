# 实验报告 08 · 深化：Uplift 建模与个性化发券名单

> 脚本：`exp_uplift.py` ｜ 交易级样本 42720 ｜ 接续 §11.3 DML 群体 CATE

## 1. 动机
§11.3 的 DML 给出**群体级**促销因果效应（CATE）。但发券是**个体级**决策——"该给谁发券"。本模块做个体处理效应(ITE)估计并生成可执行发券名单。

## 2. 方法演进（三次迭代，择优）
| 方法 | AUUC 提升比 | 问题 |
|---|---|---|
| T-learner（双模型）| 1.23× | 处理组协变量分布不同→外推偏差，群体均值与 DML 矛盾 |
| R-learner（残差伪结果）| 1.72× | 伪结果 Ỹ/T̃ 除法不稳定，ITE 出现极端值 |
| **DR-learner（AIPW 双重稳健）** | **1.70×** | ✅ 有界稳定、双重稳健、与 DML 自洽 |

**DR-learner**（最终采用）：交叉拟合 (\mu_0,\mu_1,e(X))，AIPW 伪结果
$$\psi=\mu_1(X)-\mu_0(X)+\frac{T(Y-\mu_1)}{e(X)}-\frac{(1-T)(Y-\mu_0)}{1-e(X)}$$
再用 LightGBM 拟合 (\tau(X)=E[\psi\mid X])。倾向 (e) 裁剪到 [0.02,0.98] 控制权重。

## 3. 结果
- **AIPW ATE = −0.21 元/笔**，与 §11.3 DML 的 −2.5 同为负——二者一致地推翻朴素 +7.18 的假象，确认促销对笔均收入因果效应近零/负。
- **Qini/AUUC = 14071，达随机发券基线的 1.70 倍**：按预测 uplift 降序发券，可比随机发券多捕获 70% 的增量销售额（`up_Qini提升曲线.png`，曲线显著凹于基线上方）。
- **发券名单**（`coupon_targeting_list.csv`）：1046 名顾客 uplift>0 → 建议发券；其余 1565 名 uplift≤0 → 改用权益/不打折，避免毁利。

各群体应发券比例：高价值稳定型 58.1% 最高、生鲜高频型 22.1% 最低——印证「促销资源应精准投放到个体 uplift 为正者，而非按群体一刀切」。

## 4. 业务价值
将 §11.3「不该全员撒券」的定性结论，落地为**可执行的个体发券清单 + 量化的 1.70× 增益**。配合 §11.3：对 uplift≤0 的高价值顾客用会员权益替代降价。
