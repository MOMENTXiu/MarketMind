# 1. 数据预处理（`data_preprocessing.py`）

**目的**：逐列筛查并清洗，产出标准化明细。
**方法**：错位行还原、去重、非法日期订正、退货标记、单位归一（59→43）、单价/规格填补、促销二值化、时间特征派生。

**控制台输出：**
```text
[CSV] cleaned_sales_data.csv  shape=(42813, 21)
[CSV] 数据质量报告.csv  shape=(11, 3)
[CSV] 单位映射表.csv  shape=(19, 2)
=== 清洗后概览 ===
记录数: 42813 | 顾客: 2611 | 商品: 6140 | 正向购买: 42720
```

**数据质量报告（全量）：**

| 处理项目 | 数值/统计 | 说明 |
|---|---|---|
| 原始记录数 | 42816 | — |
| 错位行修复 | 2 | 规格含逗号致右移，已按金额/单价还原数量 |
| 完全重复行去除 | 3 | 剩余 42813 |
| 非法日期订正 | 2 | 如非闰年 2/29→当月最后一天 |
