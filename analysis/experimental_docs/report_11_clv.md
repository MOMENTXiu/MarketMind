# 实验报告 11 · 广度：顾客生命周期价值 CLV（BG/NBD + Gamma-Gamma）

> 脚本：`exp_clv.py` ｜ 交易=（顾客×日）｜ 观测期截至 2025-04-30

## 1. 模型
非合同制零售场景的概率 CLV 框架：
- **BG/NBD**：由 (frequency, recency, T) 建模顾客未来购买次数与「存活」概率；
- **Gamma-Gamma**：建模客单价（与购买频次近似独立）；
- **CLV**：结合二者预测未来 90 天价值（月贴现 1%）。

## 2. 校准/留出验证（1–3 月校准 / 4 月留出）
| 指标 | 数值 |

| 留出期实际总购买 | 1667 |

预测总量仅高估 3.4%，分箱预测 vs 实际紧贴 y=x（`clv_预测校准.png`），模型**校准良好、可信**。

## 3. CLV 结果
- 全体顾客未来 90 天 CLV 合计 **≈23.7 万元**；
- **Top10% 高价值顾客贡献 56.9% 的 CLV** —— 与分群阶段帕累托效应一致；
- 输出 `clv_predictions.csv`：含历史复购次数、预测购买数_90天、存活概率、预测客单价、CLV_90天。

典型高 CLV 顾客有两类：高频中客单（如 50 次复购、存活概率 0.997）与低频高客单（如 1 次但客单价 665 元、存活概率 0.768），前者靠黏性、后者靠大额。

## 5. 业务价值
- CLV 为营销预算分配提供依据：**高 CLV + 高存活概率 → 维系投入**；**高 CLV + 低存活概率 → 优先挽回**（与流失模型 D、Uplift 模块 B 联动）；
- 升级了思路文档的静态 RFM/M 指标为**前瞻性概率预测**。
