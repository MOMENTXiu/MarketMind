# 实验报告 09 · 广度：顾客流失预测与 SHAP 可解释

> 脚本：`exp_churn.py` ｜ 特征窗 1–2 月 / 标签窗 3–4 月 ｜ 1774 名 1–2 月活跃顾客

## 1. 任务定义
对 1–2 月有购买的顾客，预测其 3–4 月是否**完全无购买（流失）**。这是对思路文档「流失预警型」群体的预测化升级——从事后画像到**事前概率预警**。

- 特征（截至 2/28）：最近购买间隔、购买频次、消费金额、记录数、客单价、商品数、小类数、促销金额占比、生鲜占比、类目熵、活跃跨度。
- 模型：LightGBM（class_weight=balanced），7:3 分层切分。

## 2. 结果
| 指标 | 数值 |
|---|---|
| 流失率（基线）| 0.517 |
| ROC-AUC | **0.781** |
| PR-AUC | 0.738 |
| 准确率 | 0.72 |

混淆矩阵（`ch_混淆矩阵.png`）与 ROC 曲线（`ch_ROC曲线.png`）显示模型在均衡阈值下对留存/流失双类 F1≈0.72，显著优于随机。

## 3. SHAP 可解释（`ch_SHAP蜂群.png`、`ch_SHAP重要性.png`）
流失驱动因子重要性排序：**促销金额占比 > 购买频次 > 客单价 > 最近购买间隔 > 活跃跨度**。
- **购买频次/活跃跨度高 → 流失概率低**（黏性强）；
- **最近购买间隔大 → 流失概率高**（已显疏远）；
- 促销金额占比的方向性显示促销参与度高的顾客留存更稳。

## 4. 业务落地
输出 `churn_predictions.csv`（全量顾客流失概率 + 高/中/低风险等级）。其中**高风险 839 人**，叠加消费金额上 40% 筛出 **218 名高价值高风险顾客**作为**优先召回名单** → 配合 §11.3/Uplift 发券（对其中 uplift>0 者发召回券，其余用权益挽留）。
