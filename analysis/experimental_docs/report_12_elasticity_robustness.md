# 实验报告 12 · 深化：价格弹性(DML) 与鲁棒性检验

> 脚本：`exp_elasticity_robust.py`

## G. 需求价格弹性（DML 去偏）

模型：(\ln Q = \varepsilon\cdot\ln P + g(X)+\epsilon)，(\varepsilon) 即价格弹性。用 DML（连续处理，交叉拟合）在控制类目（中类/小类 one-hot）、促销、时间、商品热度后估计去偏弹性。

| 范围 | 价格弹性 ε | 95%CI | 解读 |
|---|---|---|---|
| 整体 | **−0.25** | [−0.26,−0.24] | 缺乏弹性 |
| 日配 | −0.31 | [−0.36,−0.26] | 相对最敏感 |
| 休闲 | −0.28 | 显著 | 零食可促销走量 |
| 粮油 | −0.23 | 显著 | |
| 蔬果 | −0.13 | 显著 | 刚需 |
| 酒饮 | −0.14 | 显著 | |
| 洗化/家居/针织/冲调 | ≈0 | — | 极不敏感 |

**结论**：各品类需求**普遍缺乏价格弹性**（|ε|<1，远离单位弹性线），符合超市刚需特性。**日配/休闲/粮油相对最敏感**→适合促销走量；**洗化/冲调/家居近零弹性**→可适度提价提升毛利。这与 §11.3 DML「促销整体侵蚀毛利」结论互补：在低弹性品类降价既不走量又损毛利。

> 说明：估计已强控类目（仅用品类内价格变动），属保守的「纯价格响应」，不含跨品类替代效应。

## C1. 关联规则统计显著性（卡方 + Bonferroni）
对 43 条小类级规则做 2×2 卡方独立性检验，Bonferroni 校正阈值 α=0.05/43≈1.16e−3：
- **36/43 条（83.7%）统计显著**；
- 最强：袋装薯片↔其他膨化点心 χ²=1044（p<1e−228）、花果↔根茎 χ²=760、根茎↔叶菜 χ²=566 等全部极显著。

证明核心关联规则**并非偶然共现**，可放心用于组合促销。`rule_significance.csv`。

## C2. 聚类稳定性（bootstrap ARI）
25 次 bootstrap 重抽样重拟合 GMM(k=7)，与参考分群计算调整兰德指数：
- **平均 ARI = 0.48 ± 0.12**。

属**中等稳定**——与「零售顾客是双模态连续谱」的核心发现一致：精细 7 分群的边界会随样本波动，但宏观结构稳定。实务建议：营销定位用 7 群（软概率容错），需高稳定的硬决策（如权益资格）用 2 个宏观群。
