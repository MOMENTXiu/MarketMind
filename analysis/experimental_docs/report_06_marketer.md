# 实验报告 06 · 营销者侧决策（含 DML 因果推断）

> 脚本：`exp_marketer.py`、`causal_dml.py` ｜ 输入：清洗明细 + 顾客画像 + 分群 + HUIM/关联规则

## 1. 群体营销价值识别（§11.1）
对 7 个顾客群体构建价值评价矩阵 `[群体总销售额, 人均消费, 购买频次, 复购率, 促销响应率, 流失风险(负向)]`，CRITIC 自动赋权 + TOPSIS 排序 → `segment_value_rank.csv`。

高价值稳定型 / 高价值稳定型2 价值排名居前，二者合计贡献约 75% 销售额，是会员运营核心。

## 2. 商品组合促销决策（§11.2）
以 HUIM 高效用组合为候选，匹配小类级规则的 confidence/lift，构建 `[支持度,置信度,提升度,总效用,促销提升度,可触达篮数]`，TOPSIS → 组合价值得分。

Top 组合：**叶菜+新鲜蛋品+花果**（价值 0.651，促销提升度 3.11）等生鲜跨品类组合 → 跨品类联动·第二件折扣。输出 `bundle_strategy.csv`。

## 3. 促销响应分析：朴素 vs DML 因果（§11.3）★ 创新点

**核心发现——符号反转**：

| 方法 | 促销对笔均销售额的影响 |
|---|---|
| 朴素均值差（混淆）| **+7.18 元/笔** |
| **DML 去偏 ATE** | **−2.50 元/笔**（95%CI [−4.95, −0.04]，显著）|

朴素比较显示「促销笔均金额更高」，但这是**混淆假象**——被促销的商品本就集中在高价值品类/情境。双重机器学习（交叉拟合 + Robinson 部分线性，控制品类、单价、价格分位、时间、顾客 RFM 等混淆变量）揭示：促销实际上使**笔均收入下降约 2.5 元**（折扣效应盖过走量效应）。

**算法**（`causal_dml.py`，合成数据验证可还原已知效应，详见自检）：
```
Y=θ·T+g(X)+ε,  T=m(X)+ν           # T=是否促销, Y=销售金额
交叉拟合: ĝ=E[Y|X], m̂=E[T|X] (LightGBM, 5折 out-of-fold)
残差: Ỹ=Y−ĝ, T̃=T−m̂ ;  θ̂=ΣT̃Ỹ/ΣT̃²
影响函数估计方差 → 95%CI
```

**异质效应 CATE（按群体）**：
| 群体 | CATE (元/笔) | 显著 | 解读 |
|---|---|---|---|
| 促销敏感型2 | **+1.17** | 边际 | 唯一正响应，促销资源应优先投放 |
| 高价值稳定型2 | −0.54 | 否 | 促销无益 |
| 高价值稳定型 | **−1.45** | 是 | 促销纯让利、侵蚀毛利，应少打折多权益 |
| 生鲜高频型 | −1.93 | 否 | — |
| 类目集中型 | **−3.25** | 是 | 促销显著毁利 |
| 流失预警型 | −12.1 | 否（CI 宽）| 样本少 |
| 促销敏感型 | −18.0 | 否（CI 宽）| 样本少 |

**营销含义**：除「促销敏感型2」外，对多数群体（尤其高价值稳定型）的无差别促销会**侵蚀毛利**。应停止「全员撒券」，转向**精准投放促销敏感群体 + 对高价值群体用权益替代降价**。这正是思路文档朴素 PromoLift 无法给出的结论。

## 4. 品类经营分析（§11.4）
对 15 个大类计算 `[销售贡献, 复购率, 促销依赖度, 增长率, 人均金额]` → 五类经营策略：

| 类型 | 代表大类 | 策略 |
|---|---|---|
| 核心稳定类目 | 蔬果(复购0.81)、休闲、粮油、酒饮、肉禽 | 保供应·会员权益 |
| 促销依赖类目 | 日配(促销依赖0.46)、洗化、冲调、针织 | 控促销成本·提毛利 |
| 增长潜力类目 | 水产(增长+225%) | 增曝光·交叉推荐 |
| 长尾弱势类目 | 家居、熟食、文体、家电、烘焙 | 打包清理·降库存 |

输出 `category_operation_strategy.csv`。
