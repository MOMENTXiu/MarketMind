# 实验报告 04 · 商品关联与高效用组合

> 脚本：`exp_association.py` ｜ 购物篮 8942 个（顾客×日，正向购买），均篮 4.74 件，88.5% 多品篮

## 1. 实验设计（思路文档 §8、§14.2）

| 实验 | 方法 | 阈值 | 目的 |
|---|---|---|---|
| E1 | FP-Growth 商品级 | sup≥0.003, conf≥0.20, lift≥1.10 | 个性化加购推荐 |
| E2 | FP-Growth 小类级 | sup≥0.01, conf≥0.25, lift≥1.15 | 稳定共购/组合促销 |
| E3 | 高效用项集挖掘 HUIM 小类级 | sup≥0.002, 效用≥中位 | 高金额组合识别 |
| E4 | 层级融合（商品级优先→小类级兜底→中类级解释）| — | 多粒度策略 |

**购物篮**：`B_{u,d}=` 顾客 u 在日期 d 购买的去重集合。

**FP-Growth 伪代码**：
```
TransactionEncoder one-hot → fpgrowth(min_support, max_len)
association_rules(metric=confidence) → 过滤 lift≥t, |后项|=1 → 按 lift,conf 排序
```

**HUIM 伪代码（捕获低频高价值）**：
```
低支持度 FP-Growth 生成候选项集 X (|X|≥2)
对每个 X: U(X)=Σ_{篮b⊇X} Σ_{i∈X} amount(i,b)
按 min_utility(总效用中位数) 过滤, 按总效用降序取 Top
```

## 3. 关键洞察
- **生鲜强共购社区**：叶菜/根茎/花果/菌菇/鲜猪肉/鲜调味/新鲜蛋品形成密集共购网络（图13），是「家庭买菜」的典型组合 → 适合生鲜满减、联动陈列。
- **零食配对**：薯片↔膨化点心 提升度 11.6，强独立配对 → 第二件折扣。
- **HUIM 的价值**：根茎+鲜猪肉支持度仅 0.04（FP-Growth 易忽略），但**篮均效用 22.4 元最高** → 低频高价值组合，HUIM 成功识别，验证思路文档创新点四。
